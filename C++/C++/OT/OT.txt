1.절차지향 프로그래밍 언어
-C언어는 절차지향 언어 (순차적으로 실행 ,함수 위주)

- 구조체(C언어)
- 함수 생성 불가능 (C++에서는 가능)
- 접근 제어자를 사용하지 않았을 경우 public
- 구조체 상속 불가능(C++에서는 가능)
- 구조체 템플릿 사용 불가능(C++에서는 가능)

-오버로드 / 오버라이드 사용 불가능
- 참조변수 사용 불가능 ex) int &a =b

2. 객체 지향 프로그래밍 언어
- C++가 객체지향 언어 (객체위주로 실행 , 클래스 위주)
- 캡슐화 , 상속, 당형성

- 캡슐화 : 중요한 데이터에 접근하지 못하돌고 설정해 보호하는 것
-public / protected / private

-상속 : 자식(하위) 클래스가 부모클래스의 멤버를 상속 받는것

- 다형성 : 객체를 다양한 방법으로 동작할 수 있게 만드는것
- 오버로드 , 오버라이드

클래스(C++)
-생성자 / 소멸자 , 함수 생성가능
-접근제어자를 사용하지 않았을 경우 private
- 클래스 상속 가능
- 클래스 템플릿 사용 가능
- 오버로드 / 오버라이드 사용가능
- 참조변수 사용가능
